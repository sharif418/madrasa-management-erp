     1тЖТ# Madrasa Management ERP тАФ Project Worklog
     2тЖТ
     3тЖТ## Project Overview
     4тЖТA production-ready, multi-tenant SaaS Madrasa Management ERP built with Next.js 16.
     5тЖТDesigned for deployment on Coolify. Each madrasa (tenant) gets isolated data via row-level
     6тЖТtenancy, with full RBAC, multi-language support (Bangla, English, Arabic), and a complete
     7тЖТfeature set covering academics, Hifz tracking, Shariah-compliant finance, and more.
     8тЖТ
     9тЖТ## Tech Stack (Adapted from agent's recommendation)
    10тЖТ- **Framework**: Next.js 16 with App Router (single-page SPA at `/`)
    11тЖТ- **Language**: TypeScript 5
    12тЖТ- **Styling**: Tailwind CSS 4 + shadcn/ui (New York)
    13тЖТ- **Database**: Prisma ORM + SQLite (multi-tenant via `tenantId` row-level isolation)
    14тЖТ- **Auth**: NextAuth.js v4 (credentials provider, phone-based)
    15тЖТ- **i18n**: next-intl (Bangla, English, Arabic with RTL support)
    16тЖТ- **State**: Zustand (client), TanStack Query (server)
    17тЖТ- **Charts**: Recharts
    18тЖТ- **Icons**: Lucide React
    19тЖТ
    20тЖТ## Architectural Decisions
    21тЖТ1. **Single-page app at `/`**: All user-facing UI is on `/`. Navigation handled via
    22тЖТ   Zustand store (no Next.js routing for views). API routes under `/api/*`.
    23тЖТ2. **Multi-tenancy via Row-Level Isolation**: Every domain table has `tenantId`.
    24тЖТ   A Prisma client extension auto-injects `where: { tenantId }` for tenant queries.
    25тЖТ3. **Modular folder structure**:
    26тЖТ   - `src/modules/<domain>/` тАФ feature modules (each with api, components, store)
    27тЖТ   - `src/components/` тАФ shared UI components
    28тЖТ   - `src/lib/` тАФ core utilities (db, auth, i18n, tenant)
    29тЖТ   - `src/i18n/` тАФ translations (bn, en, ar)
    30тЖТ4. **File size limit**: No file over 300 lines. Split into focused modules.
    31тЖТ5. **Three languages** with full UI translation + RTL for Arabic.
    32тЖТ
    33тЖТ## Feature Modules
    34тЖТ1. **Landing Page** тАФ SaaS marketing, login, tenant signup
    35тЖТ2. **Dashboard** тАФ Analytics, charts, recent activity, quick stats
    36тЖТ3. **Students** тАФ CRUD, profiles, guardian info, class assignment, wallet
    37тЖТ4. **Teachers/Staff** тАФ CRUD, role assignment, payroll
    38тЖТ5. **Academic** тАФ Classes, sections, subjects, Qawmi/Alia curriculum
    39тЖТ6. **Hifz Tracking** тАФ Sabak, Sabaq Para, Dhor with quality ratings + reports
    40тЖТ7. **Finance** тАФ Funds (General, Lillah, Waqf, Zakat), transactions, Tamlik automation
    41тЖТ8. **Wallet** тАФ Cashless campus, top-up, internal payments
    42тЖТ9. **Attendance** тАФ Student & staff attendance
    43тЖТ10. **Exams & Results** тАФ Marks entry, report cards
    44тЖТ11. **Notices** тАФ Announcements, notifications
    45тЖТ12. **Settings** тАФ Tenant config, language, theme, RBAC roles
    46тЖТ13. **Audit Log** тАФ Track all critical changes
